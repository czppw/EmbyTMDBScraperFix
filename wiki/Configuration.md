# 配置说明

## TMDB

- `TMDB API Key`
- `TMDB API 地址`
- `TMDB 图片地址`
- `TMDB 语言`
- `TMDB 回退语言`
- `TMDB 区域`
- `启用成人内容搜索`

说明：

- `TMDB 图片地址` 可单独指向反代或 CDN
- `TMDB 回退语言` 用于补全标题、简介和人物简介

## 中文搜索增强

- `启用中文搜索增强`
- `启用标题规范化`
- `启用简繁回退`
- `启用路径与别名搜索回退`
- `最大搜索变体数`
- `输出详细搜索诊断日志`

说明：

- 标题规范化会清理分辨率、编码、帧率、音频等技术标签
- 详细搜索诊断会输出 `TMDB search selected` 和 `TMDB search rejected`

## 代理

- `启用 HTTP 代理`
- `代理地址`
- `代理端口`
- `用户名`
- `密码`
- `启用旧版全局代理 Hook`

## 自动扫描

- `启用自动增量扫描`
- `扫描间隔（分钟）`
- `检测到变更后自动刷新元数据`
- `最大重试次数`
- `媒体库`
